# 使用 How
## 1.  前往[releases](https://github.com/Beadd/MusicDownloader/releases)下载MusicDownloader.exe
下载运行即可,包含eyed3,无需pip,若无法使用请用下面的方法


## 2.  前往[releases](https://github.com/Beadd/MusicDownloader/releases)下载MusicDownloader.py
